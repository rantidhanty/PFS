Image placement guide

Main folders:
- `public/images/products` for catalog product photos
- `public/images/hero` for homepage banners
- `public/images/brands` for brand logos
- `public/images/categories` for category thumbnails

Products structure:
- `public/images/products/basketball/ring-basket-fiba/01.jpg`
- `public/images/products/basketball/ring-basket-fiba/portable/01.jpg`
- `public/images/products/basketball/ring-basket-fiba/tanam-dinding/01.jpg`
- `public/images/products/basketball/ring-basket-fiba/tanam-tanah/01.jpg`
- `public/images/products/volleyball/net-volleyball-fivb/01.jpg`
- `public/images/products/football/goal-post-11v11/01.jpg`
- `public/images/products/badminton/court-post-badminton/01.jpg`
- `public/images/products/padel/tiang-padel/01.jpg`
- `public/images/products/tennis/tiang-tennis/01.jpg`
- `public/images/products/official-equipment/kursi-volley-wasit/01.jpg`
- `public/images/products/official-equipment/kursi-badminton-wasit/01.jpg`

Naming rules:
- Use lowercase letters and hyphens only
- Use numeric sequence for gallery images (`01.jpg`, `02.jpg`, ...)
- Prefer `.webp`; use `.jpg` when needed
